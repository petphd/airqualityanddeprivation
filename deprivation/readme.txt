Index of Multiple Deprivation (IMD) - 2010 - Combined Authorities Geodata Pack: Liverpool City Region (E47000004)


+ Abstract
This geodata pack provides the CDRC IMD for the year 2010 covering the Combined Authorities: Liverpool City Region (E47000004)


+ Contents
	 - readme.txt: Information about the CDRC Geodata pack
	 - metadata.xml: Metadata
	 - tables: Folder containing the csv files
	 - shapefiles: Folder containing the shapefiles
